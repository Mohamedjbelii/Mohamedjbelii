## 🌐 AWS Cloud Deployment: **AWS_App**

### [AWS_App - Highly Available Web Application](https://github.com/Mohamedjbelii/Aws_App/tree/main/src)
![AWS Project](https://img.shields.io/badge/Cloud-AWS-orange?style=for-the-badge)

💡 **AWS_App** is a comprehensive web application deployment project hosted on AWS, designed for high availability, scalability, and secure operations. This project demonstrates the deployment of a Dockerized application consisting of a Spring Boot backend and a React frontend, all integrated with AWS services such as EC2, RDS, VPC, and Auto Scaling.

### 🛠️ **Key Features**:
- **Custom VPC Design**: Deployed the application in a VPC with public and private subnets for isolation.
- **Docker Integration**: Dockerized both frontend and backend services for easy deployment and scaling.
- **Auto Scaling & Load Balancer**: Configured **Auto Scaling Groups** to dynamically scale backend EC2 instances and an **Application Load Balancer** to distribute traffic.
- **Secure Networking**: Utilized **Security Groups**, **NAT Gateway**, and **Bastion Hosts** to ensure secure access to private resources.
- **Database**: **AWS RDS MySQL** instance used for persistent backend storage.
